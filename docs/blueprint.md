# **App Name**: TANGO CITY

## Core Features:

- City Selection: Select a major city by country and have all information updated to that city's information.
- Login: Login functionality: email login based, ability to login with Google, Facebook, and Apple.
- Milonga Registration: Milonga registration function: After registering a milonga, you can register a milonga event on the milonga page.
- Milonga Event Registration: Milonga event registration function: Select or directly enter a venue profile, directly enter a DJ or select a DJ profile, and enter information such as date, organizer, admission fee, start time, and end time to register.
- Lesson Registration: Lesson registration function: Select or directly enter the profile of the dancer who will be teaching the class, and enter the date, time, tuition fee, curriculum, etc.
- City Management: City management function
- Venue Profile Management: Venue profile management function
- DJ Profile Management: DJ profile management function
- Dancer Profile Management: Dancer profile management function
- Lesson Management: Lesson management function
- Milonga Management: Milonga management function
- Milonga Event Management: Milonga event management function

## Style Guidelines:

- Primary color: Deep red (#8B0000) to evoke passion and energy.
- Secondary color: Dark gray (#333333) for a sophisticated background.
- Accent: Gold (#FFD700) for highlighting important information and interactive elements.
- Clean and readable font for city names and details.
- Elegant and simple icons for different categories (milongas, events, lessons, etc.).
- Use a card-based layout to display tango information for each city.
- Subtle transitions when navigating between cities and details.
